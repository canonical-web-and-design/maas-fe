## Default

```jsx
<Pagination
  itemsPerPage={20}
  totalItems={100}
  paginate={() => {}}
  currentPage={3}
/>
```

## Truncated

```jsx
<Pagination
  itemsPerPage={10}
  totalItems={1000}
  paginate={() => {}}
  currentPage={34}
/>
```

## Disabled controls

```jsx
<Pagination
  itemsPerPage={20}
  totalItems={100}
  paginate={() => {}}
  currentPage={1}
/>
```
