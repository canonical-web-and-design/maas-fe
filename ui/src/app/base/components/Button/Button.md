### Base

```jsx
<Button appearance="base">Base</Button>
<Button appearance="base" disabled>
  Base disabled
</Button>
```

```jsx
<Button element="a" href="#test" appearance="base">
  Base link
</Button>
<Button element="a" href="#test" appearance="base" disabled>
  Base link button
</Button>
```
